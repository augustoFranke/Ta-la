# ADR: Localization

## Status
Accepted

## Date
2026-02-19

## Context
The product is targeted to users in Dourados, MS, Brazil.

## Decision
Keep all user-facing copy in Portuguese (pt-BR), including labels, errors, placeholders, navigation, and notifications.

## Consequences
- UX consistency for local audience.
- New features must include localization review as part of completion.
- Internal code/docs can remain in English.

- `AGENTS.md`
- `.planning/phases/01-check-in-trust-core/01-02-PLAN.md`
- `.planning/PROJECT.md`
