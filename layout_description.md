- **User Avatar:** On the top-left-corner of the screen.
- **Greeting:** Personalized text saying "Olá, {username}".

### 2. Search Bar

- **Component:** A rounded search input field with a magnifying glass icon.
- **Placeholder Text:** "Buscar bares, festas, boates..."
- **Function:** Allows the user to manually search for venues by name or location.
- **Position:** Middle of the page, on top of the "Perto de você" section.

### 3. "Perto de você" Section

- **Section Header:** "Perto de você" (Closest to you) with an arrow indicating more options can be viewed.
- **Venue Cards:** A horizontal scrolling carousel of nearby venues.
  - **Venue Image:** A photo showing the exterior of the facility.
  - **Venue Info:** Includes the name and the distance from the user ("860 m de você").
  - **Check-in Button:** A greyed-out button that says "Você está muito longe" (You are too far away). This function prevents users from checking into a venue unless their GPS shows they are physically at the location.

### 4. Bottom Navigation Bar

- **Início (Home):** Representing the home dashboard shown.
- **Explorar (Explore):** A magnifying glass icon, used for discovering new locations via map or categories.
- **Parceiros (Partners):** A grid icon, likely showing a full list of affiliated partners.
- **Minha conta (My Account):** A person icon for managing profile settings.

- **Section Header:** "Em alta" with a fire icon.
